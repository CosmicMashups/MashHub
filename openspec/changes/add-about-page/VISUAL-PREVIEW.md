## 🎭 Animation Timeline

```
Page Load:
├─ 0ms:    Page structure renders
├─ 100ms:  Header Music icon pulses
├─ 200ms:  Developer card fades in + slides up
├─ 400ms:  Sparkles icon shimmers
├─ 600ms:  Decorative icons start floating
├─ 800ms:  Category headers fade in
└─ 1000ms: Artist cards stagger in (100ms each)

Continuous:
├─ Heart icons:      Pulse every 2s
├─ Star icons:       Rotate 360° over 20s
├─ Music notes:      Bounce every 1.5s
├─ Globe:            Rotate 360° over 30s
├─ Disc3:            Rotate 360° over 25s
└─ Background icons: Parallax on scroll

On Hover:
├─ Cards:           Scale 1.05x in 200ms
├─ Youtube icons:   Scale 1.1x + color shift
├─ External icons:  Fade in + slide right
├─ Profile images:  Overlay Play icon
└─ Sparkles:        Rotate 180° in 300ms
```

## 📱 Responsive Icon Behavior

### Desktop (>1024px)
- All icons at full specified size
- Floating decorative icons visible
- Full animation suite active
- 3-4 column grid for artists

### Tablet (768px-1024px)
- Icons scale down 10%
- Floating icons reduced
- Animations slightly faster
- 2 column grid

### Mobile (<768px)
- Icons scale down 20%
- Minimal floating icons
- Simpler animations
- Single column
- Touch targets minimum 44x44px
